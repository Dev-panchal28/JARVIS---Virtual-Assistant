To create a folder named "demo", you can follow these steps:

**For Windows:**

1. Open File Explorer.
2. Navigate to the location where you want to create the folder.
3. Right-click on an empty area and select "New" > "Folder".
4. Name the folder "demo".

**For Mac:**

1. Open Finder.
2. Navigate to the location where you want to create the folder.
3. Right-click (or Control-click) on an empty area and select "New Folder".
4. Name the folder "demo".

Alternatively, you can also use the command line to create a folder:

**Using Command Prompt (Windows):**

1. Open Command Prompt.
2. Navigate to the location where you want to create the folder using the `cd` command.
3. Type `mkdir demo` and press Enter.

**Using Terminal (Mac):**

1. Open Terminal.
2. Navigate to the location where you want to create the folder using the `cd` command.
3. Type `mkdir demo` and press Enter.

The folder "demo" will be created in the specified location.
